# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ApplyTrack, please report it
responsibly:

1. **Do not** open a public issue.
2. Email **[security@example.com]** with a description and steps to reproduce.
3. Allow up to 72 hours for an initial response.
